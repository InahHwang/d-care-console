🏥 시스템 개요

D-Care V2는 치과 환자의 상담부터 치료 완료까지 전체 여정을 관리하는 시스템입니다. 환자 상태를 7단계 퍼널로 관리하여 각 단계별 필요한 액션을 명확하게 파악할 수 있습니다.

📊 대시보드

오늘의 콜백, 예약, 긴급 환자 현황을 한눈에 파악

👥 환자 관리

상태별 필터링, 검색, 상세 정보 관리

📞 통화 기록

CTI 연동 자동 녹취, AI 분석 요약

📅 일정 관리

콜백, 예약 일정 캘린더 뷰

🔄 환자 상태 흐름 (7단계 퍼널)

환자는 최초 상담부터 치료 완료까지 다음 단계를 거쳐 진행됩니다.

전화상담 → 내원예약 → 내원완료 → 치료예약 → 치료중 → 치료완료 → 사후관리
상태 설명 다음 액션
전화상담 최초 문의 후 상담 진행 중 내원 예약 유도, 콜백 설정
내원예약 내원 일정이 잡힌 상태 예약일 리마인드, 내원 확인
내원완료 병원 방문하여 상담 완료 치료 동의 유도, 치료 예약
치료예약 치료 일정이 잡힌 상태 치료 시작 확인
치료중 치료 진행 중 치료 경과 관리, 완료 확인
치료완료 모든 치료 완료 만족도 확인, 사후관리 전환
사후관리 치료 후 정기 관리 리콜 연락, 추가 치료 상담
💡 종결 처리

어떤 상태에서든 환자가 이탈하면 종결 처리할 수 있습니다. 종결 사유를 기록하여 추후 분석에 활용합니다.

📱 케이스 1: 신규 환자 등록

전화 문의, 온라인 문의 등으로 새로운 환자가 유입된 경우

📞

인바운드 전화 문의

가장 일반적
  1. 전화 수신 → CTI 자동 팝업
    CTI 연동 시 전화번호로 기존 환자 자동 조회. 신규면 등록 버튼 표시
  2. 환자 정보 등록
    이름, 전화번호, 관심 시술, 유입 경로 입력
    환자 관리 → 환자 등록 버튼
  3. 상담 진행 & 기록
    통화 종료 후 AI가 자동으로 통화 내용 요약
  4. 다음 액션 설정
    내원 예약 또는 콜백 일정 설정
    상태: 전화상담 → 내원예약 또는 콜백 설정
🌐

온라인 문의 (웹/채널)

  1. 문의 수신 알림
    네이버톡톡, 카카오 채널 등에서 문의 인입
  2. 환자 등록 & 아웃바운드 콜
    연락처 확인 후 환자 등록, 상담 전화 진행
    상담타입: 아웃바운드 선택
  3. 상담 결과 기록
    상담 내용 및 다음 액션 설정

🔔 케이스 2: 콜백 관리

상담 후 추가 연락이 필요한 경우 (재상담, 예약 확인 등)

🔄

콜백 예정 환자 처리

  1. 오늘 콜백 목록 확인
    대시보드 또는 일정 관리에서 오늘 콜백 예정 환자 확인
    대시보드 → 오늘 콜백 카드 클릭
  2. 환자에게 전화
    환자 카드에서 전화 아이콘 클릭하여 발신
  3. 통화 결과에 따른 처리
    연결됨: 상담 후 상태 변경
    부재중: 다음 콜백일 재설정
⚠️ 지연/노쇼 관리

콜백일이 지난 환자는 대시보드에 빨간색으로 표시됩니다. 환자 목록에서 "노쇼/지연" 필터로 확인할 수 있습니다.

콜백 사유별 처리

콜백 사유 처리 방법
단순 부재중 다음 콜백일 설정 (1~2일 후)
고민 중 온도 체크 후 콜백일 설정, 관심도에 따라 간격 조절
가격 문의 이벤트/혜택 정보 안내 후 재상담
일정 조율 내원 가능 일정 확인 후 예약

🏥 케이스 3: 내원 관리

내원 예약된 환자의 방문 및 상담 관리

📅

정상 내원

예약일에 방문
  1. 내원 확인
    예약 시간에 환자 도착 확인
  2. 상태 변경: 내원완료
    환자 상세 페이지에서 상태를 "내원완료"로 변경
    상태 변경 → 내원완료 선택 → 내원일 입력
  3. 내원 상담 기록
    대면 상담 내용 기록 (+수동입력 → 내원 선택)
  4. 치료 동의 여부 확인
    동의: 치료예약 상태로 변경, 치료 시작일 설정
    보류: 콜백 설정하여 추후 연락
❌

내원 노쇼

예약일에 미방문
  1. 노쇼 확인
    예약 시간 경과 후 미방문 확인
  2. 연락 시도
    환자에게 전화하여 사유 확인
  3. 후속 처리
    재예약: 새로운 예약일 설정
    이탈: 콜백 설정 또는 종결 처리

💉 케이스 4: 치료 관리

치료 동의 후 치료 진행 및 완료 관리

✅

치료 진행

  1. 치료 시작
    상태를 "치료중"으로 변경, 치료 시작일 기록
    상태 변경 → 치료중 선택
  2. 금액 정보 입력
    예상 금액, 실제 결제 금액, 수납 상태 기록
  3. 치료 경과 기록
    치료 메모에 진행 상황 기록
  4. 치료 완료
    모든 치료 완료 시 상태를 "치료완료"로 변경
    상태 변경 → 치료완료 선택
💰 수납 상태 관리

미수납: 결제 전
부분수납: 일부 결제 완료
완납: 전액 결제 완료

치료 완료 후 사후관리

치료 완료된 환자는 "사후관리" 상태로 전환하여 리콜 관리합니다.

  • 정기 리콜 연락 (3개월, 6개월, 1년)
  • 만족도 확인 및 후기 요청
  • 추가 치료 상담 (다른 시술 권유)

🚪 케이스 5: 종결 처리

환자가 이탈하거나 더 이상 관리가 필요 없는 경우

🔒

환자 종결

  1. 종결 사유 확인
    환자 이탈 사유 파악
  2. 종결 처리
    환자 상세 → 종결 버튼 클릭
    종결 사유 선택 (타병원, 비용, 연락두절 등)
  3. 종결 기록 저장
    종결 사유가 "기타"인 경우 상세 내용 입력

종결 사유 분류

종결 사유 설명
타병원 선택 다른 병원에서 치료 결정
비용 문제 비용 부담으로 치료 포기
치료 불필요 상담 결과 치료가 필요 없음
연락 두절 여러 차례 연락 시도 실패
지역/거리 문제 병원 위치로 인한 이탈
기타 위 사유에 해당하지 않는 경우 (상세 내용 입력)
🔄 종결 환자 재활성화

종결된 환자가 다시 문의하면 상태를 변경하여 다시 관리할 수 있습니다.

👥 환자 목록 필터

  • 기간 필터: 1개월, 3개월, 6개월, 전체, 직접 설정
  • 상태 필터: 전화상담, 내원예약, 내원완료, 치료예약, 치료중, 치료완료, 사후관리
  • 긴급 필터: 노쇼/지연, 오늘, D-3 이내
  • 검색: 이름 또는 전화번호로 검색

📝 상담 기록

환자별로 상담 이력을 관리합니다.

  • 자동 기록: CTI 연동 통화 시 AI가 자동 요약
  • 수동 입력: +수동입력 버튼으로 전화상담/내원 상담 기록

⚙️ 설정 (관리자 전용)

  • 카테고리 관리: 상담타입, 유입경로, 관심시술 항목 관리
  • 사용자 관리: 직원 계정 생성 및 권한 관리
  • 초대 관리: 신규 직원 초대 링크 발급

📱 모바일 사용

태블릿이나 스마트폰에서도 접속 가능합니다. 반응형 디자인으로 화면 크기에 맞게 조정됩니다.

⚠️ 주의사항

• 환자 정보 수정 후 반드시 "저장" 버튼을 클릭하세요.
• 상태 변경 시 날짜 정보를 정확히 입력하세요.
• 종결 처리는 신중하게 진행하세요 (복구 가능하지만 이력이 남습니다).

자주 묻는 질문

Q. 환자 상태를 잘못 변경했어요.

A. 환자 상세 페이지에서 다시 상태를 변경할 수 있습니다. 변경 이력은 모두 기록됩니다.

Q. 콜백일을 변경하고 싶어요.

A. 환자 상세 페이지의 "예정일" 영역에서 날짜를 수정할 수 있습니다.

Q. 이전 상담 내용을 보고 싶어요.

A. 환자 상세 페이지 하단의 "상담 이력" 섹션에서 모든 상담 기록을 확인할 수 있습니다.

Q. 한 환자가 여러 치료를 받아요.

A. "여정" 기능을 사용하세요. 환자 상세 페이지에서 새 여정을 추가하여 각 치료를 독립적으로 관리할 수 있습니다.
