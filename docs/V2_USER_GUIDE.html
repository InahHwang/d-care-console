1. 시스템 개요

D-Care Console V2는 환자가 전화하는 순간부터 치료 완료 후 사후관리까지 전체 과정을 관리하는 통합 시스템입니다.

핵심 특징

📞 CTI 연동

전화가 오면 자동으로 발신번호 인식 및 환자 정보 팝업

🤖 AI 통화 분석

통화 녹음을 AI가 자동 분석하여 환자 정보 추출

📊 7단계 퍼널 관리

전화상담부터 사후관리까지 체계적인 환자 여정 관리

🔔 자동 리콜/콜백

놓치면 안 되는 환자를 자동으로 알림

메뉴 구성

메뉴 설명
대시보드 오늘의 상담 현황을 한눈에 확인
통화 기록 모든 수신/발신 통화 기록 및 AI 분석 결과
환자 관리 등록된 환자 목록 및 상세 정보 관리
일정 관리 콜백, 리콜, 감사전화 일정 관리
소개 관리 환자 소개 네트워크 및 감사 관리
리포트 통계 및 분석 데이터
설정 시스템 설정 (상담타입, 시술 목록 등)

2. 환자 관리 사이클 (7단계 퍼널)

모든 환자는 아래 7단계 퍼널을 통해 관리됩니다. 각 단계에서 다음 단계로 전환되거나, 중간에 종결될 수 있습니다.

7단계 퍼널 시각화

1. 전화상담
2. 내원예약
3. 내원완료
4. 치료예약
5. 치료중
6. 치료완료
7. 사후관리

각 단계별 설명

단계 상태 설명 주요 액션
1 전화상담 전화로 상담 중인 환자 콜백 예약, 내원 유도
2 내원예약 내원 날짜가 잡힌 환자 예약일 리마인드
3 내원완료 병원에 방문한 환자 치료 상담, 견적 제시
4 치료예약 치료 날짜가 잡힌 환자 예약일 리마인드
5 치료중 현재 치료 진행 중 치료 일정 관리
6 치료완료 모든 치료가 끝난 환자 만족도 확인, 리콜 등록
7 사후관리 정기 관리 대상 주기적 리콜 발송

이탈 (종결) 케이스

퍼널 진행 중 언제든 종결 처리가 가능합니다:

  • 타병원 선택
  • 치료 의향 없음
  • 연락 두절
  • 기타 사유

종결된 환자도 다시 연락이 오면 복구하여 퍼널에 재진입시킬 수 있습니다.

3. 실제 업무 흐름

시나리오 1: 신규 환자 전화 수신

Step 1. 전화 수신 및 CTI 알림
  • 전화가 오면 화면 우측에 CTI 패널이 팝업됩니다
  • 발신번호가 표시됩니다
  • 기존 환자면 이름과 정보가 함께 표시됩니다
Step 2. 통화 후 AI 분석 확인

통화 종료 후 약 20초 뒤, AI가 통화 내용을 자동 분석합니다.

[통화 기록] 메뉴에서 분석 결과를 확인하세요:

  • 분류: 환자 / 거래처 / 스팸 / 기타
  • 관심 시술: 임플란트, 교정, 충치치료 등
  • 관심도: Hot(높음) / Warm(중간) / Cold(낮음)
  • 통화 요약: 3줄 요약
Step 3. 환자 등록

통화 기록에서 [환자 등록] 버튼을 클릭하여 환자를 등록합니다:

  • 환자 이름 입력
  • 상담타입 선택 (초진/재진)
  • 관심 시술 확인/수정
  • 유입경로 선택 (네이버, 지인소개 등)
  • 나이, 주소 (선택)
Step 4. 콜백 예약 (필요시)

즉시 예약이 어려운 경우:

  • 환자 상세 페이지에서 [콜백 예약] 클릭
  • 연락할 날짜/시간 설정
  • [일정 관리] 메뉴에서 확인 가능

시나리오 2: 기존 환자 상태 변경

내원예약 잡기
  • [환자 관리] 메뉴에서 해당 환자 클릭
  • 상단 상태 퍼널에서 [내원예약] 클릭
  • 예약 날짜 선택 후 저장
내원완료 처리

환자가 실제로 내원했을 때:

  • 환자 상세 페이지에서 [내원완료] 상태로 변경
  • 상담 결과 입력: 예상 금액(견적), 관심 시술, 상담 메모
치료 진행

치료예약 → 치료중 → 치료완료 순으로 상태를 변경합니다:

  • 각 단계에서 실제 금액, 메모 등 기록
  • 치료완료 시 수납 금액 입력

시나리오 3: 콜백/리콜 관리

오늘의 콜백 확인
  • [일정 관리] 메뉴 → [콜백] 탭
  • 오늘 할 콜백 목록 확인
  • 각 항목에서 바로 전화 걸기 가능
  • 통화 후 완료/미완료 처리
리콜 설정 (치료 완료 환자)

치료 완료된 환자에게 주기적으로 리콜을 보냅니다:

  • [일정 관리] → [리콜] 탭
  • 치료 종류별 리콜 주기 설정 (예: 스케일링 → 6개월 후)
  • 자동으로 리콜 대상 환자 목록 생성

시나리오 4: 소개 환자 관리

소개자 등록

새 환자가 기존 환자 소개로 왔을 때:

  • 새 환자 등록 시 유입경로를 "지인소개"로 선택
  • [소개 관리] 메뉴에서 소개 관계 등록
  • 소개자 → 피소개자 연결
감사 전화
  • [소개 관리]에서 감사 전화 대기 목록 확인
  • 소개자에게 감사 전화 후 완료 처리

4. 메뉴별 상세 기능

4.1 대시보드

오늘의 현황을 한눈에 파악합니다.

제공 정보

  • 오늘 총 통화 건수
  • 신규 환자 수
  • 예정된 콜백 수
  • 부재중 전화 수
  • 이번 달 매출 현황 (예상/실제)
  • 주의가 필요한 환자 (노쇼 위험, 콜백 누락 등)

출근 직후 대시보드에서 오늘 할 일을 파악하세요. 알림 카드를 클릭하면 해당 환자로 바로 이동합니다.

4.2 통화 기록

모든 수신/발신 통화를 기록하고 AI가 분석합니다.

주요 기능

  • 날짜별 통화 기록 조회
  • 분류별 필터 (환자/거래처/스팸/기타)
  • 수신/발신 필터
  • AI 분석 결과 확인 및 수정
  • 바로 환자 등록

AI 분석 정보

항목 설명
분류 환자인지 아닌지 자동 판별
관심 시술 통화에서 언급된 시술 (임플란트, 교정 등)
관심도 Hot(높음) / Warm(중간) / Cold(낮음)
요약 통화 내용 3줄 요약

AI가 잘못 분류한 경우 [분류 변경] 버튼으로 수동 수정할 수 있습니다.

4.3 환자 관리

등록된 모든 환자를 관리합니다.

필터 기능

  • 기간 필터: 최근 1개월/3개월/6개월/전체
  • 상태 필터: 퍼널 단계별 필터
  • 긴급 필터: 오늘 예약/콜백 누락/노쇼 위험

환자 카드 정보

  • 이름, 전화번호
  • 현재 상태 (퍼널 단계)
  • 관심 시술
  • 관심도 (온도 아이콘)
  • 마지막 연락일
  • 예상/실 금액

환자 상세 페이지

  • 기본 정보 조회/수정
  • 상태 변경 (퍼널 이동)
  • 통화 이력
  • 상담 기록
  • 금액 정보 (예상/실제/수납상태)
  • 메모

4.4 일정 관리

콜백, 리콜, 감사전화를 관리합니다.

3개 탭

탭 용도
콜백 전화상담 후 재연락이 필요한 환자 관리
리콜 치료 완료 후 정기 검진 안내
감사전화 환자를 소개해준 분께 감사 인사

4.5 소개 관리

환자 소개 네트워크를 관리합니다.

제공 정보

  • 총 소개 건수
  • 감사 전화 완료/대기
  • Top 소개자 랭킹

기능

  • 소개 관계 등록
  • 감사 전화 완료 체크
  • 소개자/피소개자 환자 상세로 이동

4.6 리포트

통계 및 분석 데이터를 제공합니다.

  • 일별/주별/월별 통화 현황
  • 퍼널별 전환율
  • 유입경로별 분석
  • 매출 현황
  • 상담사별 실적

4.7 설정

시스템 설정을 관리합니다.

  • 상담 타입 관리 (초진/재진 등)
  • 관심 시술 목록 관리
  • 유입경로 목록 관리
  • 리콜 주기 설정
  • 사용자 관리

5. 자주 묻는 질문 (FAQ)

AI 분석이 틀렸을 때는 어떻게 하나요?
통화 기록에서 해당 건 클릭 → [분류 변경] 버튼으로 수동 수정할 수 있습니다.
이미 등록된 환자가 다시 전화하면?
CTI에서 자동으로 기존 환자 정보를 표시하고, 통화 기록이 해당 환자에 자동 연결됩니다.
환자 상태를 잘못 변경했을 때는?
환자 상세 페이지에서 다시 올바른 상태로 변경하면 됩니다. 상태 변경 이력이 모두 기록되어 있어 추적이 가능합니다.
종결된 환자가 다시 연락오면?
환자 상세 페이지에서 [복구] 버튼으로 다시 활성화할 수 있습니다.
콜백 예약을 놓쳤을 때는?
대시보드의 주의 환자 카드에 "콜백 누락" 환자가 표시됩니다. 클릭하여 바로 처리하세요.
같은 환자가 여러 번 등록됐을 때는?
전화번호가 같은 환자는 중복 등록이 방지됩니다. 만약 중복이 발생했다면 하나를 종결 처리하세요.
